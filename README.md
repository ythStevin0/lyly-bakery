# Lyly Bakery Lamongan

Toko roti dan kue terkemuka di Lamongan dengan berbagai produk segar dan halal.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Products

- Roti Manis
- Donat
- Cake
- Pastry
- Roti Gurih
- Jajan Pasar
- Artisan Bread
